1)var: This is like a legacy box from the 90s. It doesn't really respect boundaries and can pop up where we don't expect it, which often leads to bugs.

let: This is the modern, polite way to store something. It stays exactly where we put it. we can change what's inside the box whenever we want.

const: Short for "constant". Use this when we want to "lock" the box. Once we put something in, we can't replace it with something else.

2)The Spread Operator is basically a shortcut in JavaScript that uses three dots (...) to unpack or spread out the contents of a collection into individual elements.


3)forEach(): we go through each item and do something , but we don't get anything back at the end. It's just an action.

map(): we take a list, do something to every single item, and it gives we a new list with the results.

filter(): we look at wer list and only keep the ones that pass a certain test. It returns a new, shorter list.
