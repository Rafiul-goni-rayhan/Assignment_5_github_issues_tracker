GitHub Issues Tracker 🐛
Overview:
GitHub Issues Tracker is a user-friendly web application designed to help developers and teams manage project issues efficiently. It features a secure login system, a comprehensive dashboard for issue tracking, and detailed issue views.

Main Technology:

React

Tailwind CSS v4

DaisyUI v5

Key Features:

Authentication: Secure login page with demo credentials (admin / admin123).

Issue Dashboard: View a complete list of project issues with tags (Bug, Enhancement, Help Wanted) and priorities (High, Medium, Low).

Search & Filter: Easily filter issues by status (All, Open, Closed) and search for specific issues using the search bar.

Detailed View: Interactive popup/modal to view detailed issue descriptions, assignees, and publish dates.

Responsive Design: Fully optimized for mobile, tablet, and desktop views.

Dependencies:

daisyui

 lucide-react
